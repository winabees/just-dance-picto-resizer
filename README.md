# just-dance-picto-resizer
A small interface made with python to resize pictoes for jd modding and more.

Requirements:
- Python 3 installed on your system.
- Pillow (PIL) library (Install via terminal: pip install pillow).

How to Use:
1. Run main.py.
2. Click "Select Folder" and choose the folder where your pictograms are located. (Note: All pictograms must be in .png format).
3. In the size entry, type your target dimensions using the WIDTHxHEIGHT format (e.g., 256x256).
4. Click the "Export" button.

Your newly resized pictograms will appear inside the "output" folder. Enjoy!
